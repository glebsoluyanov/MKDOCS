# SPELLBOOKS, AMULETS & BLUEPRINTS

Humans, Elves and Dark Elves need spellbooks to learn certain basic spells. Orcs need amulets for the same reason. Dwarven Artisans need blueprints to create golems. All spellbooks, amulets and blueprints weigh 120.
